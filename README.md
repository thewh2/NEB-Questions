# NEB Practice Questions Hub 📘

A premium, interactive web application designed for NEB (National Examination Board) Class 12 students in Nepal. This platform provides a centralized, high-performance interface for practicing previous year questions and model papers across multiple subjects.

![NEB Questions Banner](https://img.shields.io/badge/NEB-Practice_Questions-6366f1?style=for-the-badge&logo=javascript)

## 🚀 Key Features

- **✨ Ultra-Smooth UI/UX**: Professional glassmorphism design with premium animations and 3D page-turning effects.
- **🔍 Deep Search Engine**: Fast, tag-level search filtering by subject name, year, or even specific file contents.
- **🖐️ Interactive Touch Tracking**: Native mobile feel with real-time hand-gesture tracking and image dragging.
- **📄 Save as PDF**: Instantly bundle multi-page question papers into a single, high-quality PDF document.
- **⭐ Favorites System**: Save difficult or important questions to a persistent "Favorites" tab for quick revision.
- **📅 Dual-Year Support**: High-confidence exam prep with both English (AD) and Nepali (BS) year formats.
- **📱 Mobile First**: Fully responsive design optimized for smartphones, tablets, and desktops.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla CSS3 (Custom Design System), JavaScript (ES6+)
- **Libraries**: [jsPDF](https://github.com/parallax/jsPDF) for client-side PDF generation.
- **Typography**: Google Fonts (Outfit).
- **Icons**: Fluent Emojis & Custom CSS Graphics.

## 📂 Project Structure


## 💻 Getting Started

To run this project locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/thewh2/NEB-Questions.git
   ```

2. **Open the project**:
   Simply open `index.html` in any modern web browser. No server-side setup required!
